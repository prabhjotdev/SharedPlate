## Responsive Breakpoints

Although mobile-first, the app should gracefully scale:

```
Mobile (default):    Full width, stacked layout
Tablet (md:768px):   2-column grid for library, centered forms
Desktop (lg:1024px): Max-width container, 3-column library grid
```

**Tailwind Config Suggestion:**
```css
/* Container for desktop */
.container {
  @apply max-w-lg mx-auto;
}

/* Library grid responsive */
.recipe-grid {
  @apply grid grid-cols-2 md:grid-cols-3 lg:grid-cols-4 gap-4;
}
```
